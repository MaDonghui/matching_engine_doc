Detailed Description

template<typename T>
class SparseSet< T >

A SparseSet can hold arbitrary value with self-increasing size
Internal storage is organised in term of Pages for improved space-complexity
Page is a static array with customizable size that must be power of 2

Time-Complexity

  • retrieve O(1)
  • write O(1), worst case O(n) if page pool needs to be resized
  • remove O(1)
    Template Parameters
    TAny type of element to be stored
